About mAnkInd

mAnkInd is an interactive experience where a human player faces off against an AI in a real-time conversational duel. The objective is to determine whether the opponent is human or AI before the time runs out.

Features

  • Real-time Duel
  • Dynamic AI
  • XP System
  • Multiple Game Modes

Tech Stack

Frontend: HTML, CSS (Tailwind), JavaScript

Backend: Python (FastAPI)

AI: Gemma 2 (Hugging Face)
